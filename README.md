### Description
**EcoBandit** is an interactive, gamified application designed to encourage sustainability on campus. By leveraging AI image recognition, the platform allows students to scan recyclable materials and earn points for their eco-friendly actions. 

The idea is that the accumulated points can be exchanged for tangible items and campus perks, made possible through a collaboration with *the Student Government's Student Welfare committee*.

### Features (Under Development)
- **AI Recognition:** Utilizes machine learning to scan, identify, and categorize recyclable items.
- **Gamified Rewards:** Earn points for every valid scan.
- **Time Limit:** A student can only scan 1 item every 5 minutes.
- **Daily Cap:** Max 100 points per day per student.
- **GPS Geofencing:** The "Upload" button is disabled unless the student's phone GPS matches the coordinates of the "Campus".
